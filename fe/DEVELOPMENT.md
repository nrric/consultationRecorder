# Development Guide

## Prerequisites

| Item                           | Version |
| -------------------------------| ------- |
| [Node](https://nodejs.org/en)  | ≥ 20    |
| [PNPM](https://pnpm.io/)       | ≥ 8     |

## Setup

### 1. Install Dependencies

```bash
pnpm install
```

### 2. Run service

```bash
pnpm dev
```
